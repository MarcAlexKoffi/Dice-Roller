# Dice Roller App

Une application Android simple développée avec "Kotlin" et "Jetpack Compose" permettant de lancer un dé virtuel. À chaque pression sur le bouton, une nouvelle face de dé s'affiche aléatoirement.

# Fonctionnement du projet

- Au lancement de l'application, une image représentant une face de dé (de 1 à 6) s'affiche.
- Un bouton `"Roll"` permet de générer un nouveau chiffre entre 1 et 6 à chaque clic.
- L’image du dé se met automatiquement à jour en fonction du chiffre obtenu.
- L’interface est centrée verticalement et horizontalement.


# Technologies utilisées

- Kotlin
- Jetpack Compose
- Android Studio version Meerkat Feature Drop 2024.3.2 Patch 1
